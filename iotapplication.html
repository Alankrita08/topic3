Summary of Activities

ACTIVITY 1:

In this activity, my knowledge about IoT applications enhanced. This activity included some questions that had to be answered in a group. While doing this activity, I learnt that IoT applications are the applications that integrate hardware and software and are connected with internet to transmit data. The main fetaures that an IoT application should have are scalability, accessibility, connectivity and a good communication between the devices and the application. There are alot of examples of IOT applications and some of them are smart watch, smart cars, smart home applications like smart lighting system, smart air conditioning system and smart tracking systems.

ACTIVITY 2:

In this activity, I learnt how something can be made smart using Internet of Things. This activity included some questions that had to be answered in a group for the scenario of the smart building. While doing this activity, I learnt that smart means to make something efficient using technology to ease the work. A smart building should have sensors, actuators and buttons to sense the environment and do the things accordingly. A smart building should be economically efficient, highly secure and should use appropriate use of resources. Some of the smart building scenarios include automated lighing system so that less energy to be consumed, automated temperature maintaining system, high level security system using different kinds of sensors. ALl these systems make the environment smart.
